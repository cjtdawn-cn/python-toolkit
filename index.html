🐍 Python Toolkit

10个实用Python脚本 · 零依赖 · 即开即用

🚀 快速开始

git clone https://github.com/cjtdawn-cn/python-toolkit.git
cd python-toolkit
python3 file_organizer.py --help

每个脚本独立运行，无外部依赖（除标注外）

📁 file_organizer.py

按类型自动整理文件夹
python3 file_organizer.py ~/下载

✏️ batch_rename.py

批量重命名文件
python3 batch_rename.py ./photos trip_

📱 qr_generator.py

二维码生成器
pip install qrcode[pil]
python3 qr_generator.py "https://..."

🔐 password_gen.py

随机安全密码生成
python3 password_gen.py 10

📊 text_counter.py

字数统计+词频分析
python3 text_counter.py article.txt

🖼️ image_resizer.py

批量调整图片大小
pip install Pillow
python3 image_resizer.py 800 600

🔄 csv_to_json.py

CSV转JSON格式
python3 csv_to_json.py data.csv

🕷️ web_scraper.py

简单网页爬虫
pip install requests bs4
python3 web_scraper.py URL

📄 pdf_merger.py

PDF合并工具
pip install PyPDF2
python3 pdf_merger.py *.pdf

📝 markdown_to_html.py

Markdown转HTML
python3 markdown_to_html.py README.md
